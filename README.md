# Rajadarshini Saravanan – Portfolio

Portfolio site for **Rajadarshini Saravanan** – Bharatanatyam artist, movement researcher, yoga & wellness educator, and founder of TAT Yoga Studio.

## Live site

Hosted on GitHub Pages. After pushing, enable Pages in **Settings → Pages** (source: main branch, root) to get:

`https://<your-username>.github.io/Rajadarshini-Saravanan/`

## Contents

- `index.html` – Single-page portfolio (HTML, CSS, JS)
- `profile.png` – Profile photo
- `gallery-1.jpg` … `gallery-7.jpg` – Gallery images

## Custom domain

To use a GoDaddy (or other) domain: add the domain under **Settings → Pages → Custom domain**, then set the DNS records at your registrar as GitHub instructs.
